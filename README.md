# peer-extra


* peer_app.py - API Header

To work with codework

* codework_get.py - Get student's answers
* codework_update.py - Post scores, comments to student's answers
* codework_to_tsq.py - Post scores to t-sq. You will need to download T-sq's grades.csv file.

To add students to the peer website

* roster.py - Add students from a csv file

Deprecated files (probably)

* create_roster.py
* pr_to_tsq.py

Others

* export_tsq.py - Upload comments, scores from projects to T-sq
